## 🚀 快速开始

### 在线阅读
访问我们的在线文档：https://maim-with-u.github.io/Mailauncher-docs/

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/MaiM-with-u/Mailauncher-docs.git
cd Mailauncher-docs

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建文档
npm run build
```
